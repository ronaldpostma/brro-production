brro-production.php - Plugin with custom functions and styles for theworldofwalking.nl


php/brro-global-functions.php - Contains global functions used across the entire website.
php/brro-admin-functions.php - Contains functions related to the WordPress admin area.
php/brro-gallery-field.php - Contains functions for managing custom gallery fields.
php/brro-woocommerce-functions.php - Contains custom functions for WooCommerce.
php/brro-search-functions.php - Contains custom functions for search customization.
php/brro-homepage-functions.php - Contains custom functions for the homepage.
php/brro-inspiratie-functions.php - Contains custom functions for the 'Inspiratie' section.
php/brro-evenement-functions.php - Contains custom functions for the 'Events' section.


css/brro-twow-global.css - Global CSS file for styling various parts of the website.
css/brro-checkout-cart.css - CSS file for styling the checkout and cart pages.
css/brro-wp-admin.css - CSS for wp-admin for all users
css/brro-admin-users.only.css - CSS for wp-admin for specific admin users
css/brro-editor-users.only.css - CSS for wp-admin for specific editor users


js/brro-global-script.js - Global JavaScript file for adding interactivity to the website.
js/brro-checkout-cart.js - JavaScript file for the checkout and cart pages.
js/brro-single-product.js - JavaScript file for single product pages.
js/brro-product-tag-archive.js - JavaScript file for product tag archives
js/brro-wp-admin.js - JavaScript file for wp-admin pages.
